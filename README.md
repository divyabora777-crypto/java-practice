# Java Practice
This repository contains Java programs that I am writing while learning
and practicing core Java concepts.
## ATM Simulation
A simple console-based ATM program written in Java.
### Features
- Check account balance
- Deposit money
- Withdraw money
- Menu-based user interaction
- Basic input validation
The program runs in the console and allows users to select different
options until they choose to exit.
